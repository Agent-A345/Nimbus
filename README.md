## 🌤️ Nimbus — Live Weather Dashboard

A modern, visually immersive Weather Web App built using HTML, CSS, and JavaScript, providing real-time weather data, multi-city comparison, animated UI, and intelligent weather insights using Open-Meteo APIs.

## 🚀 Features
* **Real-Time Weather Data** – Fetches live weather data including temperature, humidity, wind, and pressure.
* **City Search with Suggestions** – Search cities with auto-suggestions powered by geocoding API.
* **Current Location Detection** – Detects user location using geolocation for instant weather updates.
* **Multi-City Comparison** – Compare weather across multiple cities in a grid view.
* **7-Day Forecast** – Displays upcoming weather trends with temperature highs and lows.
* **24-Hour Temperature Chart** – Visualizes hourly temperature trends using canvas charts.
* **Smart Weather Insights** – Provides contextual suggestions based on weather conditions.
* **Dynamic Weather Backgrounds** – Changes UI theme based on weather (rain, snow, sunny, etc.).
* **Animated Particle Effects** – Adds rain, snow, and atmospheric effects dynamically.
* **Live Local Time Display** – Shows real-time local time of selected city.
* **Unit Toggle (°C / °F)** – Switch between Celsius and Fahrenheit instantly.
* **Theme Toggle (Dark / Light)** – Toggle between dark and light UI modes.
* **Recent Searches** – Stores and displays recently searched cities.

## 🎮 How to Use
**Search Weather**
* Enter city name in search bar
* Select from suggestions or press Enter

**Use Current Location**
* Click location button
* App fetches weather based on GPS

**Multi-City Mode**
* Switch to Multi-City view
* Add multiple cities
* Compare weather instantly

**Customize View**
* Toggle temperature unit (°C / °F)
* Switch theme (Dark / Light)
